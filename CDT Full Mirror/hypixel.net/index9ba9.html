Jun 7, 2022 Hypixel now supports Minecraft 1.19



Hey everyone!

Minecraft 1.19 is out! The Hypixel Server has now been updated to support the new version.

As such, at this time, we do support the following versions:
1.8, 1.12, 1.14, 1.15, 1.16, 1.17, 1.18, and 1.19!

If you come across any sort of exploits, bugs, issues, or anything else, please report it to our bug report section.

Finally, as always, I'd like to remind everyone that Hypixel is better played on the 1.8.9 version of Minecraft if you tend to play PVP games, as is usually the case for any Minecraft Servers out there.

Regarding Hypixel SMP 1.19

As usual, there is more work involved to support 1.19 fully in SMP. We aim to support 1.19 in SMP very soon and we'll be working as fast as we can to support it!

UPDATE: We have enabled 1.19 support for SMP, please note that servers may be slower to start for a few hours/days due to missing optimizations that will be added back as soon as possible!
